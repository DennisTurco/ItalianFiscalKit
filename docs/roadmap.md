# Roadmap

This page tracks planned features and improvements for ItalianFiscalKit.
For the full list of open issues and discussions, see the [GitHub Issues](https://github.com/DennisTurco/CodiceFiscale/issues) page.

## 1. Planned Features

| Feature | Status | Notes |
|---|---|---|
| Omocodia normalization | 🔜 Planned | Normalize omocodia variants back to canonical form |
| `ValidationResult` pattern | 🔜 Planned | Return structured errors from `FiscalCodeValidator` |
| `FiscalCodeMatcher` | 🔜 Planned | Verify a CF matches given personal data |
| `FiscalCodeExtensions` | 🔜 Planned | Extension methods on `string` and `FiscalCodeData` |

## 2. Released

See the [Changelog](../CHANGELOG.md) for what has already shipped.

## 3. Suggest a Feature
